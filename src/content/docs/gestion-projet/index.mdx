## Principes de conduite du projet

**Documentation continue** - chaque configuration déployée, chaque décision technique est documentée dans la foulée sur DocScalia et versionnée via Git. Aucune tâche critique n'est considérée terminée sans sa documentation associée.

**Binômage** - aucune compétence critique n'est détenue par une seule personne. Le RACI garantit qu'un Consulté est toujours en mesure de reprendre une tâche en cas d'indisponibilité d'un membre.

**Security by Design** - les règles de durcissement ANSSI et les GPO de sécurité sont appliquées dès l'installation de chaque service, pas en phase finale.

**Revue hebdomadaire** - chaque samedi (en fonction des disponibilités de chacun), l'équipe fait le point sur l'avancement du sprint, les blocages et les risques. Le compte-rendu est commité sur DocScalia dans les 24h.
