## Fichier de configuration

Le fichier `astro.config.mjs` à la racine du projet contient toute la configuration.

## Options principales

### `title`

Le titre du site affiché dans la barre de navigation.

### `sidebar`

La structure de la barre latérale. Vous pouvez définir les entrées manuellement ou utiliser `autogenerate` pour générer automatiquement la sidebar à partir d'un dossier.

### `locales`

Configuration des langues supportées par le site.
