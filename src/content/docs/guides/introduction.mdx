## Présentation du client <Badge text="BAI26.2" variant="danger" />

**FreeMotions** est une entreprise spécialisée dans la location de trottinettes et voitures électriques, implantée à Paris avec une antenne à Lyon. Avec un chiffre d'affaires de 15 millions d'euros et une cinquantaine de collaborateurs, FreeMotions fait face à une problématique critique : l'obsolescence de son Système d'Information.

L'infrastructure actuelle ne répond plus aux exigences de sécurité, de performance et de scalabilité nécessaires à l'activité de l'entreprise, notamment dans un contexte d'expansion multi-site.

## Périmètre du projet

L'ESN **Scalia** a été mandatée pour concevoir et déployer l'intégralité du nouveau SI de FreeMotions. Le projet couvre les axes suivants :

<CardGrid>
  <Card title="Parc informatique" icon="laptop">
    Renouvellement de 50 postes (40 fixes, 10 portables) avec déploiement zero-touch via Intune/Autopilot, BitLocker et Defender.
  </Card>
  <Card title="Réseau & Sécurité" icon="warning">
    VPN site-to-site Paris-Lyon (pfSense), segmentation VLAN, DMZ, IDS/IPS Suricata, pfBlockerNG et géoblocage.
  </Card>
  <Card title="Identité & Annuaire" icon="setting">
    Active Directory (Windows Server 2025), LAPS, SSSD pour Linux, Azure AD Connect, MFA et serveur bastion.
  </Card>
  <Card title="Supervision & Audit" icon="magnifier">
    Centreon/Zabbix avec dashboards "Météo des Services", audit de vulnérabilités OpenVAS, PingCastle pour l'AD.
  </Card>
</CardGrid>

## Stack technique

| Domaine | Technologies |
|---|---|
| Virtualisation | Proxmox VE |
| Réseau / UTM | pfSense, OPNsense, pfBlockerNG |
| Annuaire | Active Directory, SSSD, Azure AD Connect |
| Postes de travail | Windows 10/11 Pro, RDS|
| Messagerie & Collaboration | Microsoft 365, Exchange Online |
| Stockage | Serveur de fichiers (FSRM, déduplication), Ceph/Nextcloud |
| Sauvegarde | Veeam Backup (règle 3-2-1) |
| Supervision |Zabbix, Wazuh, Grafana |
| Gestion de parc | GLPI |
| Sécurité | Nmap, Nessus, OpenVAS, PingCastle |
| Web |  Nginx, HaProxy |

## Planning projet

Le projet se découpe en **5 phases** sur une durée de 9 mois :

| Phase | Période | Livrables |
|---|---|---|
| Lancement & Conception | Déc. 2025 - Jan. 2026 | Note de cadrage, architecture cible |
| Maquettage | Fév. - Mars 2026 | Environnement de test, validation des choix techniques |
| Déploiement & Migration | Avr. - Juin 2026 | Mise en production, migration des données |
| Sécurisation & Audits | Juil. 2026 | Pentests, durcissement, conformité RGPD |
| Recette & Clôture | Août 2026 | PV de recette, documentation finale |

<Aside type="danger" title="Échéance">
Livraison finale prévue le **23 août 2026**. Budget avec écart toléré inférieur à 5%.
</Aside>
